# BeMyValentine — walentynkowa aplikacja webowa

Prosty statyczny frontend: pytanie z przyciskami `Tak.` i `nie`. Przyciski reagują — przycisk `nie` ucieka i zmniejsza się przy kolejnych kliknięciach aż do 1 piksela.

Jak uruchomić lokalnie:

1. Otwórz plik [public/index.html](public/index.html#L1) w przeglądarce.

Wdrożenie (GitHub Pages):

- W repozytorium dodałem workflow GitHub Actions, który publikuje zawartość folderu `public` na branchu `gh-pages` po pushu do `main`.
- Aby wdrożyć: zacommituj i wypchnij zmiany do `main`. Akcja automatycznie wypchnie stronę na branch `gh-pages` i (po chwili) będzie dostępna przez GitHub Pages.
